# PATIENT bundle

File này chứa riêng phần giao diện cho tác nhân: patient

Bao gồm:
- src/modules/patient
- src/routes/patientRoutes.jsx
- các thư mục shared cần thiết để mở trong VS Code

Cách dùng:
1. giải nén
2. chạy npm install
3. chạy npm start

Ghi chú:
- đây là bundle tách riêng theo tác nhân
- có thể copy các file này vào project chính hoặc dùng để tham chiếu/tái cấu trúc
