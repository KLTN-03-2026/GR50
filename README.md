# ADMIN bundle

File này chứa riêng phần giao diện cho tác nhân: admin

Bao gồm:
- src/modules/admin
- src/routes/adminRoutes.jsx
- các thư mục shared cần thiết để mở trong VS Code

Cách dùng:
1. giải nén
2. chạy npm install
3. chạy npm start

Ghi chú:
- đây là bundle tách riêng theo tác nhân
- có thể copy các file này vào project chính hoặc dùng để tham chiếu/tái cấu trúc
